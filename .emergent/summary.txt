<product_requirements>
The Ceibaa platform is a web-based competitive exam preparation tool featuring Solo Practice and real-time Live Battles.

**Core Requirements:**
1.  **Question Content Management:** The system must source all quiz questions from Google Sheets. An admin-facing SheetManager UI () is required to map specific Google Sheet URLs to an Exam, Subject, Topic, and Sub-topic.
2.  **Quiz Functionality:**
    *   Users must be able to select an exam, topic, and sub-topic to start a quiz.
    *   In Solo Practice, users should be able to configure the number of questions (e.g., 10, 25, 50, 100) before starting.
    *   The system must correctly handle quizzes for subjects that do not have a corresponding Google Sheet mapped, failing gracefully.
3.  **Advanced Content Rendering:** The application must accurately render complex mathematical and chemical notations (e.g., fractions, exponents,  bonds, ) within question and answer text. This was implemented using the KaTeX library, which requires source text in the Google Sheets to be formatted with standard LaTeX delimiters ( for inline,  for display).
4.  **Battle Room:** Live battles must support sub-topic selection to ensure the correct question set is loaded for all participants.
5.  **System Stability:** The underlying services (FastAPI backend, React frontend, Node.js battle-server) must be stable, with robust connectivity.
</product_requirements>
